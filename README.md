# ATM Management System

## Description

A Python-based ATM Management System that supports account creation, secure login using PIN, balance inquiry, deposit, withdrawal, money transfer, PIN change, and transaction history. Data is stored using file handling to maintain account information permanently.

## Features

* Create Account
* Login with Account Number and PIN
* Check Balance
* Deposit Money
* Withdraw Money
* Transfer Money
* Change PIN
* View Transaction History
* Logout
* File Handling

## Technologies Used

* Python

## Concepts Used

* Variables
* Functions
* Dictionaries
* Loops
* Conditional Statements
* File Handling
* Authentication
